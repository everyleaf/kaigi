kaigi
=====

Teaser site for every.rb


## environment

* ruby 1.9
* Sass

## setup

* clone this repo
* create gemset (if neccessary)
* `bundle install`

## development

* compile sass with `rake build`
* you can run `rake watch` to auto compile sass
** sass directory is lib/
** the rake task compiles lib/*.sass to assets/css/*.css
